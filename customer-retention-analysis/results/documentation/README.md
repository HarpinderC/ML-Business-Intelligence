# Project Documentation

This folder contains manually created documentation files that provide 
context and explanations for the project.

## Files

### Data Quality Analysis
- `data_quality_report.md` - Comprehensive data quality assessment
  - Quality issues identified and resolved
  - Filtering pipeline documentation
  - Final dataset statistics
  - Known limitations

### Feature Documentation
- `feature_definitions.md` - All 30 features documented
  - Calculation formulas
  - Interpretation guides
  - Feature importance rankings

### Business Recommendations
- `business_recommendations.md` - Executive summary and strategic priorities
  - 4 strategic priorities (Save At All Costs, Loyalists, etc.)
  - Campaign economics and ROI projections
  - Implementation timeline
  - Success metrics

### Compliance & Responsible AI
- `gdpr_compliance_notes.md` - GDPR and Responsible AI documentation
  - Data protection principles
  - Right to explanation (SHAP)
  - Automated decision-making governance
  - Fairness and transparency considerations

### ROI Calculator
- `intervention_roi_calculator.py` - Standalone Python script
  - Calculate break-even retention rate
  - ROI scenario analysis
  - Can be run independently from notebooks
